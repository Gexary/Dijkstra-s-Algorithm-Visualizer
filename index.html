Dijkstra's Algorithm Visualizer

Graph visualization using Dijkstra's Algorithm

Sélectionner une colonne pour afficher le chemin le plus court

Animation Speed

Guide d'utilisation :

  • Pour créer un nœud, double-cliquez sur le graphe.
  • Pour créer un lien, double-cliquez sur le nœud de départ et sélectionner le nœud d'arrivée.
  • Pour sélectionner plusieurs nœuds, maintenez la touche Shift enfoncée.
  • Pour supprimer un nœud ou un lien, cliquez sur Suppr et la sélection sera supprimée.
  • Pour lancer la visualisation, cliquez sur le bouton "Start" puis cliquez sur le graphe.
